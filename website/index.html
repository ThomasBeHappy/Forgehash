ForgeHash

Configurable, memory-hard password hashing for benchmarking and cryptographic experimentation. First variant: ForgeHash-B3 (BLAKE3).

Experimental. ForgeHash is experimental cryptographic software. Not for production password storage. Prefer Argon2id, scrypt, bcrypt, or platform password APIs.

  • AlgorithmForgeHash-B3
  • Encoded idforgeh
  • Versionv=1
  • Encoded form$forgeh$v=1$m=<memoryKiB>,t=<iterations>,p=<parallelism>$<salt-b64>$<hash-b64>

Base64 is RFC 4648 without padding. Parameter order is always m,t,p.
